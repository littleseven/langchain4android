## 1. Agent 交互

### 1.1 交互模式

PicMe 提供两种交互模式，用户可自由切换：

| 模式 | 入口 | 适用场景 |
|------|------|----------|
| **对话模式** | 底部输入栏 | 复杂指令、连续调节、探索发现 |
| **快捷模式** | 右侧控制栏 | 高频单操作、快速调节 |

### 1.2 对话模式

**界面布局**
- 输入栏位于屏幕底部，半透明毛玻璃背景
- 点击展开对话面板，显示历史消息
- 支持语音输入（按住说话）和文字输入

**Agent 反馈规范**
- **确认类**：「已为你调高美颜强度」
- **澄清类**：「你想调节磨皮还是美白？」
- **建议类**：「光线较暗，是否开启夜景模式？」
- **错误类**：「抱歉，我没听懂，请再说一次」

**多轮对话示例**
```
用户：调高美颜
Agent：已为你调高美颜强度（磨皮+20）
用户：再高一点
Agent：已将磨皮提升至 50
用户：换一个滤镜
Agent：已切换至「胶片金」滤镜
```

### 1.3 支持的意图

| 意图类别 | 示例指令 | Agent 响应 |
|----------|----------|------------|
| **美颜调节** | 「磨皮 50」「美白高一点」「恢复默认」 | 参数确认 + 实时预览 |
| **滤镜切换** | 「换个冷调」「复古风格」「原图」 | 滤镜名称 + 视觉切换 |
| **相机控制** | 「拍照」「翻转镜头」「拉近一点」 | 操作执行 + 状态反馈 |
| **参数查询** | 「当前美颜多少？」「有什么滤镜？」 | 数值回复 / 列表展示 |
| **闲聊** | 「今天天气怎么样」「你会什么」 | 友好回复 + 能力引导 |

### 1.4 体验红线

- **响应延迟**：Agent 回复首字 < 500ms
- **反馈完整**：每个用户输入必须有可见反馈
- **渐进披露**：复杂能力通过对话引导发现，不一次性罗列

## 2. 智能相机

### 2.1 美颜系统

**功能范围**
- 面部精修：磨皮、美白、瘦脸、大眼
- 妆容调节：唇色、腮红、眉毛
- 身材管理：（预留，未实现）

**交互规范**
- **入口**：右上角控制栏「美颜」按钮
- **面板**：半屏高度，底部滑出
- **实时预览**：参数调整即时生效，延迟 < 100ms
- **对比功能**：长按预览区查看原图，松手恢复

**美颜总开关**
- **默认状态**：关闭（进入相机时人脸检测不运行，节省性能）
- **开启方式**：
  - 用户点击美颜面板开关
  - 调节任意美颜参数时自动开启（`resolveNextBeautySettings`）
- **关闭方式**：
  - 用户手动关闭开关
  - 所有美颜参数归零时自动关闭
- **行为差异**：
  - **关闭时**：跳过人脸检测，仅支持滤镜/调色/风格特效
  - **开启时**：启用人脸检测，支持全部功能（含瘦脸/大眼/唇色/腮红）

**参数范围**

| 功能 | 范围 | 默认值 |
|------|------|--------|
| 磨皮 | 0-100 | 35 |
| 美白 | 0-100 | 25 |
| 瘦脸 | -50~+50 | 0 |
| 大眼 | 0-100 | 20 |
| 唇色 | 0-100 | 40 |
| 腮红 | 0-100 | 20 |
| 眉毛 | 0-100 | 15 |

### 2.2 滤镜系统

**色调滤镜**
- 徕卡经典、徕卡鲜艳、徕卡黑白
- 胶片金、胶片富士、复古
- 冷调、暖调

**风格特效（GPU 实时渲染）**
- 卡通、素描、海报、浮雕、交叉线

**交互规范**
- 横向滚动列表展示
- 风格特效独立区域，可与色调叠加
- 切换动画 < 200ms

### 2.3 拍摄交互

**快门反馈（三位一体）**
1. 触感：50ms 短震动
2. 音效：快门声同步播放
3. 视觉：50ms 黑场闪烁 + 按钮缩放

**人脸十字星**
- 锁定人脸且设备运动时显示
- 画面稳定后自动淡出
- 避免长期遮挡构图

### 2.4 技术约束

- **人脸检测**：InsightFace 2D106（默认），MediaPipe 468→106（备选）
- **渲染引擎**：自研 OpenGL ES 管线（大美丽）
- **拍照处理**：GPU 离屏渲染，预览/输出一致性

## 3. 动态相册

### 3.1 媒体查看器

**交互**
- 双指缩放（1x~4x）
- 缩放态优先平移，禁用翻页
- 回到 1x 后恢复翻页
- 展开/收起跟随手指轨迹

**工具栏**
- 顶部：关闭 | 编辑(图片) | OCR | 信息 | 分享 | 删除
- 半透明深色背景，浅色图标

### 3.2 静态图美颜编辑

**入口**
- 工具栏「编辑」按钮（仅图片）
- 长按图片直接进入

**流程**
1. 异步加载原图 + 人脸检测
2. 显示 BeautySelector 面板（同相机）
3. 拖动滑块调节参数，200ms debounce 刷新预览
4. 保存为新文件，不覆盖原图

### 3.3 OCR 文字提取

**触发**
- 工具栏「提取文字」按钮

**展示**
- 原位浮层（白色卡片 + 黑色遮罩）
- Header：标题 + 字数统计
- Content：可滚动文字
- Footer：复制 / 分享按钮

## 4. 设计系统

### 4.1 HyperOS 风格

- **大圆角**：卡片、按钮统一 28dp+
- **毛玻璃**：实时模糊 + 半透明背景
- **流体动效**：Bezier 曲线 (0.4, 0, 0.2, 1)
- **微交互**：长按触感，点击缩放 0.95x

### 4.2 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| Primary | #00E5FF | 科技蓝，强调操作 |
| Secondary | #FF4081 | 强调粉，特殊提示 |
| 深色背景 | #121212 | 深色模式 |
| 浅色背景 | #FFFFFF | 浅色模式 |

### 4.3 摄影美学

- 真实自然：避免过度美颜，保持肤色自然
- 高级质感：参考徕卡色彩科学
- 情感表达：通过滤镜传达情绪

## 5. 通用规范

### 5.1 国际化 (I18N)

**支持语言**
- 英文 (en)
- 简体中文 (zh-CN)
- 繁体中文 (zh-TW)

**术语对照**

| 简体中文 | 繁体中文 | 英文 |
|----------|----------|------|
| 相册 | 相簿 | Gallery |
| 美颜 | 美顏 | Beauty |
| 滤镜 | 濾鏡 | Filter |
| 设置 | 設定 | Settings |
| 提取文字 | 提取文字 | Extract Text |

**规范**
- 字符串资源 ID：`[feature]_[description]` 小驼峰
- 新增功能必须同步三语
- 禁止硬编码用户可见文案

### 5.2 性能要求

| 指标 | 目标 | 验证方式 |
|------|------|----------|
| 冷启动 | < 500ms | 计时器 |
| 快门延迟 | < 50ms | 高速摄像 |
| 参数跟手 | < 100ms | 人工感知 |
| 相册滚动 | 120fps | Systrace |
| GPU 处理 | 1080p < 300ms | 日志 |

### 5.3 隐私红线

- 所有 AI 处理 100% 端侧
- 零网络权限
- 用户数据仅存储在本地
- 权限申请透明，拒绝后优雅降级
