# MediaPipe Face Mesh 468点标准语义定义

## 概述

MediaPipe Face Mesh 是 Google 提供的人脸网格检测解决方案，可实时检测468个3D人脸关键点。本文档整理了468个关键点的标准语义分布，用于指导 MediaPipe 468点到106点标准的映射工作。

## 关键点区域分布

| 区域 | 关键点数 | 说明 |
|-----|---------|------|
| 左眼 | 71 | 上眼睑、下眼睑、眼角 |
| 右眼 | 71 | 上眼睑、下眼睑、眼角 |
| 嘴巴 | 40 | 上唇、下唇、嘴角 |
| 鼻子 | 52 | 鼻梁、鼻尖、鼻翼、鼻孔 |
| 眉毛 | 40 | 左眉20点，右眉20点 |
| 脸部轮廓 | 122 | 额头、脸颊、下巴、耳朵 |
| 脸部内部 | 72 | 脸部中心区域 |
| **总计** | **468** |  |

## 1. 脸部轮廓（FACE_OVAL，0-35，共36点）

MediaPipe 的 FACE_OVAL 区域包含36个点，形成闭合曲线（含额头），与106点开放轮廓不同。

| 索引 | 语义位置 | 说明 |
|-----|---------|------|
| 0 | 下巴中心 | 下巴最低点 |
| 1-7 | 左下颌 | 从下巴到左耳下 |
| 8-23 | 左脸颊到额头 | 左侧轮廓 |
| 24-35 | 额头到右脸颊 | 右侧轮廓（闭合） |

**与106点对应关系**：
- 106点轮廓为开放曲线（33点），不包含额头
- 需要选取 MediaPipe 中不含额头的轮廓点

## 注意事项

1. **坐标系**：MediaPipe 返回的坐标为归一化坐标（0-1），原点在左上角，y轴向下。

2. **前置摄像头镜像**：使用前置摄像头时，x坐标需要镜像处理：`x = 1 - x`

3. **轮廓差异**：
   - MediaPipe FACE_OVAL 为36点闭合曲线（含额头）
   - 106点标准为33点开放曲线（不含额头）
   - 映射时需要选择不含额头的 MediaPipe 轮廓点

4. **瞳孔检测**：
   - 需要开启 `refine_landmarks=True`
   - 瞳孔点索引为 468-475（478点模式）

5. **左右定义**：MediaPipe 中的"左/右"是基于**观察者视角**，与106点标准一致。

6. **深度信息**：MediaPipe 每个点包含 (x, y, z) 三维坐标，z轴以鼻尖为原点。
