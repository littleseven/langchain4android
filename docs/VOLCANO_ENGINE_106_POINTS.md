# 火山引擎106点标准语义定义

## 概述

火山引擎106点人脸关键点检测标准与商汤/Face++ 106点标准一致，是行业通用的人脸关键点检测规范。本文档定义了106个关键点的标准语义位置，用于指导 MediaPipe 468点到106点的映射工作。

## 注意事项

1. **镜像说明**：本文档中的"左/右"是基于**画面视角**（前置摄像头镜像后），即画面左侧=实际右脸，画面右侧=实际左脸。

2. **轮廓方向**：轮廓33点为开放曲线，从右鬓角（索引0）开始，经过下巴（索引16），到左鬓角（索引32）结束。

3. **瞳孔位置**：104/105与74/77位置相同，用于需要瞳孔中心点的场景。

4. **对称性**：左右脸关键点应保持对称分布，映射时需确保左右对称性。
