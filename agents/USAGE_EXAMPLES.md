# PicMe AI Agent 使用示例 - LITE Team

## 🚀 快速参考

### 通过前缀激活（4 个核心角色）
```
[PM] ...      → Product Manager (产品专家)
[RD] ...      → Software Engineer (研发工程师)
[Review] ...  → Code Reviewer (代码审查)
[QA] ...      → QA Engineer (测试专家)
```

### 常见场景

#### 1. 新功能需求
```
用户："我想添加照片压缩功能"

→ [PM] 分析用户价值
→ [RD] 提出技术方案
→ [QA] 设计测试用例
→ [Review] 验证代码质量
```

#### 2. 技术咨询
```
用户："[RD] Flow 和 LiveData 应该如何选择？"

→ [RD] 解释两者的优劣和适用场景
→ [Review] 补充最佳实践建议
```

#### 3. 代码审查
```
用户："[Review] 请帮我检查这段代码"

→ [Review] 提供评分和详细反馈
→ [RD] 回应并修复问题
```

#### 4. 测试策略
```
用户："[QA] 这个功能需要什么测试？"

→ [QA] 设计测试金字塔
→ [RD] 实现单元测试
```
