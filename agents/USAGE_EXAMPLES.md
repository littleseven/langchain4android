# PicMe AI Agent Usage Examples

## Quick Reference

### Activate by Prefix
```
[Product] ...  → Product Agent
[Engineering] ... → Engineering Agent  
[Review] ... → Review Agent
[Test] ... → Testing Agent
```

### Common Scenarios

#### 1. New Feature Request
```
User: "Add photo compression feature"

→ [Product] analyzes user value
→ [Engineering] proposes solutions
→ [Test] designs test cases
→ [Review] validates quality
```

#### 2. Technical Question
```
User: "[Engineering] How to use Flow vs LiveData?"

→ Engineering Agent explains trade-offs
→ Review Agent adds best practices
```

#### 3. Code Review
```
User: "[Review] Check this code"

→ Review Agent provides score and feedback
→ Engineering Agent responds with fixes
```
